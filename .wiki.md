# Project Summary
The **Moodlab.AI** project is a multi-tenant SaaS platform tailored for fashion businesses, utilizing artificial intelligence to provide personalized outfit recommendations, enhance customer interactions, and optimize inventory management. Key features include a virtual wardrobe, a user-friendly shopping interface, and an analytics dashboard, all designed to elevate business offerings and customer engagement while ensuring seamless integration with existing systems.

# Project Module Description
- **Backend**: Manages authentication, data operations, and business logic, supporting multi-tenancy.
- **Frontend**: Offers an intuitive interface for navigation, interactions, and tailored recommendations.
- **Data Models**: Structures data for companies, clients, products, orders, closets, curated looks, and brand rules.
- **Services**: Provides functionalities such as AI-driven recommendations, data import processes, closet synchronization, and analytics.
- **Integrations**: Facilitates CSV uploads and connections to external platforms like VTEX and Shopify.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: TypeScript, React, Vite
- **Database**: PostgreSQL
- **Deployment**: AWS Lambda, Docker

# Usage
1. **Install Dependencies**:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install` (within `app/frontend`)

2. **Build and Run**:
   - For backend: Execute `python app/backend/main.py`
   - For frontend: Execute `pnpm run build` and then `pnpm run start`
